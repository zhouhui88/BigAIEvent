# AI briefing web

Static site (no build step). Open `index.html` locally or deploy via GitHub Pages.

## Deploy (GitHub Pages)

1. Create a GitHub repo (e.g. `ai-briefing-web`).
2. Upload `index.html` and `data.json` to the repo root.
3. GitHub repo Settings → Pages → Deploy from branch (main) / root.
4. The site URL will look like: `https://<user>.github.io/<repo>/`.

## Update data

Replace `data.json` with a newly generated one.
